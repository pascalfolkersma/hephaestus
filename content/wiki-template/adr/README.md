# Architecture Decision Records

ADRs document *how* this project builds things — architectural and technical choices.

## Convention

- One file per decision, numbered with a four-digit prefix: `0001-short-slug.md`.
- Once accepted, an ADR is immutable. Superseded records stay; new ones reference them.
- Date format: `YYYY-MM-DD`.

## Status values

`Draft` | `Accepted` | `Superseded by NNNN`

## Template

```markdown
# ADR NNNN — Title

- Status: Draft
- Date: YYYY-MM-DD
- Supersedes: none

## Context

## Decision

## Consequences
```
